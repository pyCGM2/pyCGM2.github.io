PyCGM2

open source python package of the Conventional Gait Model

Nexus Applications

Automatically, PyCGM2 generates global settings and ready-to-use Vicon pipelines.

All Files are located in folder C:\\ProgramData\\pyCGM2.

Global settings (file .settings)

Role of the global setting file is to set commonly used settings of vicon pipelines

  • User is free to alter default global settings if they don't match his practice.

PyCGM2 Vicon Pipelines (file .pipeline)

Pipeline file can be imported in Vicon Nexus ( video ) for performing a set of operations.

Like Vicon Plugin Gait, modelling with pyCGM2 involves two pipelines.

  • Calibration (equivalent to static PIG pipeline) : calibrates the model (i.e. : define anatomical centres and axes) and compute static angles
  • Fitting (equivalent to dynamic PIG pipeline): recontructs segmental motion and compute both kinematics and kinetics

  • User can also force global settings ponctually by altering script argument of a Nexus python operation ( see the API Documentation to find out the list of script arguments)
