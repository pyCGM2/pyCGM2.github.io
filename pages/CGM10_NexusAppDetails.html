CGM 1.0 - Nexus App Details

Return to all available Nexus apps

This CGM version is a replication of the native Vicon Plugin gait

Vicon skeleton templates

You can use all plugin-gait vst files already embeded in Nexus. if you want to calibrate the CGM1 with the Knee Aligment Devices and medial ankle markers, you can call pyCGM2-LowerLimb_CGM1_KADmed.vst

Nexus Pipelines

Located in C:/Users/Public/Documents/Vicon/Nexus2.x/Configurations/Pipelines, you can import ready-to-use pipelines into Nexus :

  • pyCGM2-CGM1-Calibration.Pipeline
  • pyCGM2-CGM1-Fitting.Pipeline

Operation details

Figures below presented the 2 pyCGM2 Nexus Pipelines you can import into Vicon Nexus.

The main operation for either the Calibration pipeline or the Fitting pipeline is highlighted in blue. They need a c3d as input that why the first line exports your current data into a c3d.

If your acquisition session doesn t match with global settings, you need to alter the box script arguments with the pyCGM2 script argument inputs.

For instances, your subject doesn t present flat feet during the static trial, you must edit the script argument with the code: -l=0 -r=0

Script arguments

Calibration pipeline

script arguments of the operation : pyCGM2-CGM1-Calibration

Usage examples Edit the script argument cell of Vicon Nexus with:

  • -l=1 -r=0
  • -l=1 -r=0 --resetMP
  • -l=1 -r=0 --markerDiameter=14 -ps=new

Script Arguments Details Choice Command examples
‘-l’ or’--leftFlatFoot’ force left flat Foot option 1 or 0 -l=1 or --leftFlatFoot=0
‘-r’ or’--rightFlatFoot’ force right flat Foot option 1 or 0 -r=1 or --rightFlatFoot=0
‘-md’ or ‘- -markerDiameter’ marker diameter in mm -md=14 or --markerDiameter=14
‘--check’ add `cgm1` as suffix of any model outputs --check
‘-ps’ or ‘--pointSuffix’ add your custom suffix to each model output -ps=New or --pointSuffix=New
‘--resetMP’ reset computation of optional mp data --resetMP

Fitting pipeline

script arguments of the operation : pyCGM2-CGM1-Fitting

Usage examples Edit the script argument cell of Vicon Nexus with:

  • -mpfa=LRXL --proj=Proximal
  • --check

Script Arguments Details Choice Command examples
‘-md’ or ‘--markerDiameter’ marker diameter in mm -md=14 or --markerDiameter=14
‘--proj‘ moment projection Proximal

‘-mfpa‘ manual force plate assignment
Number of letter matches the number of force plate		 -mpfa=LRXL stands for :
4 forces plates with left foot asignment for force plates #1 and #4, right foot assignment for force plate #2 and no assigment for force plate #3
