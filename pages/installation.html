Installation for Non Expert

Here are installation instructions for non-experts. Experts can consult the API documentation for details about used python packages.

pyCGM2 is developped under Windows 7 on a 64 bits PC with Vicon Nexus 2.5 installed

1. Set-up your Python environment

Download and install Anaconda 32 bits - python 2.7 .

Anaconda set the python 2.7 core and install scientifical packages required for pyGGM2

2. Set-up pyCGM2

  • Download and unzip the last release of pyCGM2
  • Open a window command and go to the downloaded pyCGM2 folder containing "setup.py""
  • Type in the command windows python setup.py install

This command adds pyCGM2 into your python Library ( generally located at pathToAnacondaFolder\Lib\site-packages)

3. [Optional-For Vicon users] Set-up the python API of Vicon Nexus

  • Download and install Vicon Nexus
  • Edit the file : pathToAnacondaFolder\Lib\site-packages\PyCGM2\CONFIG.py
  • check absolute paths of Vicon Nexus

Default paths are:

NEXUS_SDK_WIN32 = 'C:/Program Files (x86)/Vicon/Nexus2.5/SDK/Win32'
NEXUS_SDK_PYTHON = 'C:/Program Files (x86)/Vicon/Nexus2.5/SDK/Python'
PYTHON_NEXUS = 'C:\\Program Files (x86)\\Vicon\\Nexus2.5\\Python'

4. Set-up OpenSim TK

  • Download and install the last 32-bits release of openSim TK
  • Open a command windows and go to the folder "C:\OpenSim ?\sdk\python"
  • run the command "python setup.py install"

5. Check you installation

Open a python console and type

import pyCGM2
pyCGM2.pyCGM2.CONFIG.checkConfig()

pyCGM2 is ready to use if you obviously get no error messages.<br>

The python command import pyCGM2 generates Global Settings, automatically placed into C:\ProgramData\pyCGM2.

Uninstall pyCGM2

  1. Go to your python Library (pathToAnacondaFolder\Lib\site-packages) and remove pyCGM2 folder manually
  2. Go to the global setting folder (C:\ProgramData\pyCGM2) and remove all.

Update pyCGM2

If you want to work with a new release. Uninstall former pyCGM 2 is recommanded. Then, you can install new release according above guidelines.
