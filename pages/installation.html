Installation for Non Expert

Here are installation instructions for non-experts. Experts can consult the API documentation for details about used python packages.

Following guidelines reproduce settings of the pyCGM2 programmer.

pyCGM2 is developped under Windows 7 on a 64 bits PC with Vicon Nexus 2.5 installed
  1. Download and install Vicon Nexus
  2. Download and install Anaconda 32 bits - python 2.7 . Anaconda set the python 2.7 core and install scientifical packages required for pyGM2
  3. Download and unzip the last release of pyCGM2
  4. [Optional]. Check Nexus paths

Edit file : pyCGM2CONFIG.py and check absolute paths of Vicon Nexus :

NEXUS_SDK_WIN32 = 'C:/Program Files (x86)/Vicon/Nexus2.5/SDK/Win32'
NEXUS_SDK_PYTHON = 'C:/Program Files (x86)/Vicon/Nexus2.5/SDK/Python'
PYTHON_NEXUS = 'C:\\Program Files (x86)\\Vicon\\Nexus2.5\\Python'
  1. Open a MS-DOS console and go to pyCGM2 folder containing setup.py
A convenient method is to double click on your pyCGM2 folder while maintaining Shift. open Command windows here will come up in the context menu
  1. Type pip install -e . ( don't forget the sign .)

This command put a link in your python path ( generally pathToAnacondaFolder\Lib\site-packages) pointing your pyCGM2 folder.

Check you installation

  • Open a python console and type
import pyCGM2
pyCGM2.pyCGM2_CONFIG.checkConfig()

pyCGM2 is ready to use if you obviously get no error messages.<br>
