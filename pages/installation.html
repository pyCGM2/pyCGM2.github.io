Installation

pyCGM2 is compatible with Microsoft Windows

Last release

pyCGM2- version 1.1.0

Installation for non expert

Beforehand, download then unzip the last pyCGM2 release.

install a python environment

Implentation was done with *the Anaconda python enviromenent. Thus we invite user to copy our environment.

Download and install Anaconda 32 bits - python 2.7

Mind you have to install a 32 bits version of anaconda, select the correct donwnload link as flag up in the following picture

During anaconda installation, the window ( see picture below) will come up.

User must enable both advanced options

For Vicon Nexus user: Configure Nexus 2

The last PyCGM2 release is configured for working with Vicon Nexus 2.6, if Nexus is installed in its default folder, ( i.e C:\\Program Files (x86)\\Vicon)

if you work with either an other version of Vicon nexus or install nexus in another folder, thus you have to configure pyCGM2.

  • within your unzipped pyCGM2 folder
  • open pyCGM2\CONFIG.py with an text editor
  • edit absolute paths of Vicon Nexus
NEXUS_SDK_WIN32 = 'C:/Program Files (x86)/Vicon/Nexus2.5/SDK/Win32'
NEXUS_SDK_PYTHON = 'C:/Program Files (x86)/Vicon/Nexus2.5/SDK/Python'
PYTHON_NEXUS = 'C:\\Program Files (x86)\\Vicon\\Nexus2.5\\Python'

Optional: set-up OpenSim

Opensim is a open Musculoskeletal analysis software. PyCGM2 exploit some feautures of its python API if you plan to work with a CGM version superior to 2.2

Install pyCGM2.

pyCGM2 propose a convenient batch installation.

  • go to your unzipped pyCGM2 folder
  • double click on installer.bat

This command will successively :

  • uninstall previous installation
  • set-up pyCGM2
  • initialize the pyCGM2 setting folder (C:\ProgramData\pyCGM2) (see also)
  • check your installation
