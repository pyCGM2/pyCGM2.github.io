PyCGM2

open source python package of the Conventional Gait Model

CGM 1.1

Overview

Whilst creating CGM 1.0 we found that there were three issue with PiG and CGM 1.1 is thus a version of the CGM in which there three issues have been addressed.

  1. The foot rotation offset in PiG was calculated relative to the torsioned tibia in static calibration but applied to untorsioned tibia in modelling dynamic data. This has been recitified
  2. The correct Cardan sequence for pelvic angles has been applied.
  3. The logic for when the medial malleolar malleolar marker is used in model calibration has been recified

Other than this the model is an exact clone of the current version of PiG.

Progress

This model has bee tested in house and has now been released for beta testing.
