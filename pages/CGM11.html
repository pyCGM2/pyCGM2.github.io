CGM 1.1

Overview

Whilst creating CGM 1.0 we identified four aspects of PiG that we felt could be addressed immediately. CGM 1.1 is thus a version of the CGM in which there three issues have been addressed.

  1. There is an inconsistency between the way that the foot rotation offset is applied during static calibration and then applied later when fitting to dynamic data whcih ahs been corrected.
  2. The correct Cardan sequence for pelvic angles has been applied.
  3. Under a specific combinations of conditions the medial malleolar malleolar marker was not being used appropriately during static calibration and teh logic for when it is used has been corrected
  4. Given that using a medial malleolar marker (if placed) has been used routinely for static calibration of the tibia for over a decade it is not at all clear why a simialr option to use a medial epicondyle marker for calibration of the femur has not been introduced. This is now possible, if the medial epicondyle marker (LMEPI, RMEPI) is found then this will be used to define the coronal plane of the knee in the same way as the KAD is used if it is present.

Other than this the model is an exact clone of the current version of PiG.
