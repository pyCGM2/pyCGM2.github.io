Calling an pyCGM2 operation from nexus do nothing

solution : check *.py file is associated with proper program (i.e. python.exe)

  • right click on a py file
  • select open with
  • select choose default program
  • select python.exe. If this command doesn t appear :
    • select browse
    • go to your python environment folder ( example C:/Anaconda2) and select python.exe

my Installation with Anaconda2 ends with a registry key Error

This is attributed to Anaconda installation, not pyCGM2 !

Therefore, contact anaconda support to mention this issue.

Nevertheless, With no certitude, this error might be due to a previous anaconda installation in 64 bits version. Forums report the following process might fix this issue

  • if you had installed Anaconda 2 (32 bits)
  • uninstall Anaconda 2 (32 bits)
  • install then uninstall Anaconda 2 (64 bits !!!)
  • re-install Anaconda 2 (32 bits)
  • reboot your PC

Eventually, the registry key of python should be updated
